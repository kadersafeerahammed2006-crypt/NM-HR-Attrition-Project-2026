## 📋 Project Overview

Comprehensive analysis of employee attrition using the **IBM HR Employee Attrition Dataset**. This project helps HR teams understand why employees leave and provides data-driven strategies to improve retention.

**Key Highlights:**
- **Overall Attrition Rate:** 16.12%
- **Strongest Driver:** Overtime (nearly 3x higher risk)
- **Best Model:** Random Forest Classifier
- **22+ Business Insights** and Strategic Recommendations
